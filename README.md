# SimpleConnexion

*[WIP] Basic "Social network" project using Express, Handlerbars and SQLite.*


## Goals

**My goal is to make a really simple social network with few objectives :**

- User profile
  - Profile pic
  - Description
- Private messages
- Friend list
- Personnal feed
  - Posts with text and pictures
  - Comments on posts 
  - Friend posts
  - Like
